First time? Step-by-step setup instructions

Mac

  1. Open the Terminal app — press ⌘ Space, type Terminal, hit Enter.
  2. If you don't have Homebrew, paste this and press Enter:
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  3. Install pipx: brew install pipx
  4. Install sdexe: pipx install sdexe
  5. Run it: sdexe — your browser opens automatically.

Windows

  1. Download and install Python 3.10+ from python.org. Check "Add Python to PATH" during install.
  2. Open Command Prompt — press Win + R, type cmd, hit Enter.
  3. Install pipx: pip install pipx
  4. Install sdexe: pipx install sdexe
  5. Run it: sdexe — your browser opens automatically.

Linux

  1. Install pipx: sudo apt install pipx (Ubuntu/Debian) or pip install pipx
  2. Install sdexe: pipx install sdexe
  3. Run it: sdexe
What happens after you run sdexe: a page opens in your browser at localhost:5001. That's the app — use it like any website, but it's running on your own machine. Close the terminal window to quit.

What it can do

Download videos & audio

  • Paste any YouTube, Instagram, TikTok, SoundCloud, Twitch, or Vimeo link
  • Save as MP3, MP4, FLAC, or WAV
  • Download full playlists or multiple links at once
  • Pick quality: 1080p, 720p, 480p for video — 320/192/128 kbps for audio
  • Thumbnail embedded as album art in audio files

PDF tools

  • Merge multiple PDFs into one — drag to reorder pages
  • Split a PDF by page ranges (e.g. "1–3, 5, 8–10")
  • Convert images (JPG, PNG, WebP) into a PDF
  • Compress a PDF to reduce its file size
  • Extract all text from a PDF to a .txt file
  • Add or remove a password from any PDF

Image tools

  • Resize an image by exact dimensions or percentage
  • Compress images in bulk — pick High, Medium, or Low quality
  • Convert between PNG, JPG, and WebP formats

File converter

  • Markdown → HTML (styled, ready to publish)
  • CSV ↔ JSON, CSV ↔ TSV
  • JSON ↔ YAML
  • XML → JSON
