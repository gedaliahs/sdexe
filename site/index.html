What's inside

Media Downloader

  • YouTube, Instagram, TikTok, and 1000+ sites
  • MP3, MP4, FLAC, WAV output
  • Batch downloads with progress tracking

PDF Tools

  • Merge, split, compress, rotate
  • Extract text and images
  • Add page numbers and passwords

Image Tools

  • Resize, crop, rotate, flip
  • Convert between PNG, JPG, WebP, GIF
  • Compress, grayscale, blur, strip EXIF

Audio & Video

  • Trim, convert, merge, compress
  • Normalize volume, change speed
  • Extract audio, video to GIF

File Converter

  • CSV, JSON, YAML, XML, TSV
  • Markdown to HTML
  • Create and extract ZIP archives

Text Tools

  • Regex, diff, find & replace, word count
  • Base64, hashing, URL encode, JWT decode
  • UUID, timestamps, password generator
