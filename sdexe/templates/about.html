sdexe

Suite for Downloading, Editing & eXporting Everything

60+ tools for media, PDF, images, audio, video, and text — all running locally on your machine. No uploads, no accounts, no cloud.

What is sdexe?

sdexe is a free, open-source toolkit that runs entirely on your computer. Install it once with pipx install sdexe, run sdexe, and get a full suite of tools in your browser at localhost:5001. No data ever leaves your machine.

Media Downloader

Download video and audio from 1000+ sites including YouTube, Vimeo, Twitter, and more. Choose format, quality, and output. Batch download multiple URLs at once.

PDF Tools

Merge, split, compress, rotate, reorder, delete pages, add watermarks, number pages, extract text and images, edit metadata, password protect, and unlock PDFs.

Image Tools

Resize, compress, convert formats, crop, rotate, flip, blur, grayscale, strip EXIF data, generate QR codes, add watermarks, create placeholders, and convert to ICO.

Audio & Video

Trim, convert, merge, reverse, change speed/pitch, normalize volume, equalize, fade audio. Trim, compress, convert, crop, rotate, resize, loop, reverse video. Record audio and screen.

File Converter

Convert between Markdown/HTML, CSV/JSON, JSON/YAML, CSV/TSV, XML/JSON. Create and extract ZIP archives.

Text Tools

Word count, find & replace, regex tester, text diff, case converter, Base64, hash generator, URL encode, JSON formatter, JWT decoder, UUID generator, timestamps, lorem ipsum, color converter, password generator.

Why sdexe?

  • 100% local — Your files never leave your machine
  • No accounts — No sign-up, no login, no tracking
  • Free & open source — MIT licensed, forever free
  • One command install — pipx install sdexe
  • 60+ tools — Everything you need in one place
  • Works offline — No internet needed (except media downloads)
